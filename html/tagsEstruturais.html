HEADER - cabeçalho

NAV - navegação, dentro tem que ter os links de navegação, o NAV sao links de rodape, porem se colocar dentro do HEADER vai subentender que é uma navegação principal, os links devem estar dentro de listas (ul/ol) > li > a

MAIN - corpo principal, apenas um por pagina, nao pode ser filha de ninguem a nao ser do body

FOOTER - rodape

SECTION - é como uma div, a section é sessoes de informacoes do conteudo que vc esta peesquisando, geralmente com um titulo, usar quando uma secao deve aparecer logicamente na estrutura de um documento, se for apenas para estilização usar (DIV)

ASIDE - é um assunto que nao tem a veer com o assunto principal da pagina, nao importante, mais usados como barras laterais, explicaçoes laterais, glossario, aviso, biografia etc, OBS: nao usa o aside para texto entre parentese ()

ARTICLE - é destinado para colocar o artigo, conteudo da pagina diferente do sect ion
